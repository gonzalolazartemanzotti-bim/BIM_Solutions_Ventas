# Proyecto: Análisis de Desempeño Comercial - BIM Solutions
## Célula de Trabajo: Arq. BIM Specialists

### 1. Descripción del Proyecto
Este repositorio contiene la implementación de un proceso administrativo integral para el análisis de ventas de servicios BIM (Building Information Modeling). Se utiliza el control de versiones para garantizar la trazabilidad de los objetivos organizacionales.

### 2. Estructura del Repositorio
* **/datos**: Contiene el dataset `ventas_bim.csv`.
* **/scripts**: Código fuente de procesamiento (`analisis_ventas.py`).
* **/resultados**: Reportes generados (indicadores y totales).
